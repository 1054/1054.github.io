AstroUnits - Units

Units and Constants

$ 1 \ \mathrm{Jy} = 10^{-26} \ \mathrm{W} \ \mathrm{m}^{-2} \ \mathrm{Hz}^{-1} $

$ 1 \ \mathrm{Jy} = 10^{-23} \ \mathrm{erg} \ \mathrm{s}^{-1} \ \mathrm{cm}^{-2} \ \mathrm{Hz}^{-1} $

$ 1 \ \mathrm{MJy} \, \mathrm{sr}^{-1} = 2.350443 \times 10^{-5} \ \mathrm{Jy} \, \mathrm{arcsec}^{-2} $

$ 1 \ \mathrm{MJy} \, \mathrm{sr}^{-1} = 2.350443 \times 10^{-5} \times \mathrm{PixScale}^2 \ \ \mathrm{Jy} \, \mathrm{pixel}^{-1} $

$ 1 \ \mathrm{W} = 10^7 \ \mathrm{erg} \ \mathrm{s}^{-1} $

$ 1 \ \mathrm{L_{\odot}} = 3.839 \times 10^{26} \ \mathrm{W} = 3.839 \times 10^{33} \ \mathrm{erg} \ \mathrm{s}^{-1} $ (without neutrino; from wiki from Foukal et al. 2006 Nature)

$ 1 \ \mathrm{M_{\odot}} = 1.98892 \times 10^{33} \ \mathrm{g}$

$ 1 \ \mathrm{pc} = 206,265 \ \mathrm{AU} = 3.085677 \times 10^{16} \ \mathrm{m} = 3.2616 \ \mathrm{ly} $

$ 1 \ \mathrm{pc} \ (\mathrm{km}/\mathrm{s})^{-1} = 3.085677 \times 10^{13} / 3.1536000 \times 10^{13} \ \mathrm{Myr} = 0.978461758 \ \mathrm{Myr} $

$ 1 \ \mathrm{pc}^{3} \times 1 \ \mathrm{H_2} \ \ \mathrm{cm}^{-3} = (3.085677 \times 10^{18})^{3} \times (3.3718047 \times 10^{-24}) / (1.98892 \times 10^{33}) = 0.04980769993 \ \mathrm{M_{\odot}} $

$ 1 \ \mathrm{Mpc}^2 = 9.52140 \times 10^{44} \ \mathrm{m}^2 = 9.52140 \times 10^{48} \ \mathrm{cm}^2 $

$ 1 \ \mathrm{L_{\odot}} \mathrm{Mpc}^{-2} = 3.839 \times 10^{33} \times 10^{23} \times 10^{-9} \times 10^{3} / (9.52140 \times 10^{48}) \ \mathrm{mJy} \ \mathrm{GHz} = 40.31970 \ \mathrm{mJy} \ \mathrm{GHz} $

$ 1 \ \mathrm{M_{\odot}} \mathrm{pc}^{-2} = \frac{1}{\mu_{\mathrm{H}}} \ 1.2488745 \times 10^{20} \ \mathrm{cm}^{-2} = \frac{1}{\mu_{\mathrm{H_2}}} \ 0.619518 \times 10^{20} \ \mathrm{cm}^{-2} $

$ 1 \ \mathrm{sr} = (3600 \times 180 / \pi) \ \mathrm{arcsec}^2 = 206265^2 \ \mathrm{arcsec}^2 = 4.25451703\times10^{10} \ \mathrm{arcsec}^2 $

$ 1 \ \mathrm{arcsec}^2 = 2.35044305 \times 10^{-11} \ \mathrm{str} $

$ 1 \ \mathrm{arcsec} = 1/3600 ^\circ = 1/3600/180\cdot\pi \ \ \mathrm{rad} = 0.004848136811 \ \ \mathrm{mrad} = 0.004848136811 \cdot d_{A} \ \ \mathrm{kpc}$

$ 1 \ k_B = 1.3806488 \times 10^{-23} \ \mathrm{Jy} \ \mathrm{K}^{-1} = 1.3806488 \times 10^{-16} \ \mathrm{erg} \ K^{-1}$

$ 1 \ h_{\mathrm{planck}} = 6.62606957 \times 10^{-34} \ \mathrm{m}^2 \ \mathrm{kg} \ \mathrm{s}^{-1} = 6.62606957 \times 10^{-27} \ \mathrm{erg} \ \mathrm{s} $

$ 1 \ \mu_{\mathrm{proton}} = 1.67262158 \times 10^{-24} \ g $

$ 1 \ \mu_{\mathrm{H_2}} = 2 \times 1.00794 \times m_{proton} = 3.3718047 \times 10^{-24} \ g $

$ \displaystyle \frac{2h\nu^3}{c^2} \frac{1}{e^{h\nu/k_{B}T_b}-1} \Omega = \frac{2h}{c^2} \frac{\nu^3}{[\mathrm{GHz}^3]} \frac{1}{e^{h\nu/k_{B}T_b}-1} \frac{\pi}{4\ln2} \frac{(\pi\theta/3600/180)^2}{[\mathrm{arcsec}^2]} = 3.926980414 \times 10^{-8} \ \frac{\nu^3}{[\mathrm{GHz}^3]} \frac{1}{e^{h\nu/k_{B}T_b}-1} \ \frac{\theta^2}{[\mathrm{arcsec}^2]} \ \mathrm{Jy} $

$ \displaystyle \frac{2h\nu^3}{c^2} \frac{1}{e^{h\nu/k_{B}T_b}-1} \Omega = \frac{2h}{c^2} \frac{\nu^3}{[\mathrm{GHz}^3]} \frac{1}{e^{h\nu/k_{B}T_b}-1} \frac{\pi}{4\ln2} \frac{(\pi\theta/3600/180)^2}{[\mathrm{arcsec}^2]} \approx 3.926980414 \times 10^{-8} \ \frac{\nu^3}{[\mathrm{GHz}^3]} \ \frac{k_{B}T_b}{h\nu} \ \frac{\theta^2}{[\mathrm{arcsec}^2]} = 8.182499051 \times 10^{-7} \ \frac{\nu^2}{[\mathrm{GHz}^2]} \ \frac{T_b}{[\mathrm{K}]} \ \frac{\theta^2}{[\mathrm{arcsec}^2]} \ \mathrm{Jy} $
