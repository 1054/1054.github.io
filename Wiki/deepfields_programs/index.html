DeepFields -- Programs

Abbrev.

  • NF = Galaxy Number Function/Counts a given wavelength
  • LF = Galaxy Luminosity Function at a given wavelength
  • IRLF = Galaxy Total Infrared Luminosity Function
  • SMF = Galaxy Stellar Mass Function
  • CSFRD = Cosmic Star Formation Rate Density
  • CIB = Cosmic Infrared Background
